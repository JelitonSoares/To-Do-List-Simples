




                      SIMPLE TO-DO LIST


PROGRAMA DEVE CONTER UM SISTEMA DE LOGIN (USUARIO E SENHA)
CADA USUARIO TERÁ SUAS PRÓPRIAS LISTAS

ACÕES:


MARCAR TAREFA COMO REALIZADA (UMA DUAS OU TRÊS)
MARCAR TODAS AS TAREFAS COMO FEITAS
APAGAR UMA TAREFA
APAGAR TODAS TAREFAS
ATUALIZAR UMA TAREFA
SETAR NÍVEL DE URGÊNCIA DE UMA TAREFA (1-URGENTE 2-ASSIM QUE POSSÍVEL 3-ADIÁVEL)
ATUALIZAR NÍVEL DE URGÊNCIA


ERROS:

MARCAR UMA TAREFA APAGADA COMO FEITA
ATUALIZAR UMA TAREFA APAGADA
ATUALIZAR NÍVEL DE URGÊNCIA DE UMA TAREFA APAGADA


                   ESTRUTURA DO BANCO

USUARIOS                         TAREFAS                                     NIVEIS_URGENCIA 

ID_USUARIO VARCHAR(5) PK         ID_TAREFA BIGINT AUTO-GENERATED PK          ID BIGINT  AUTOGENERATED FK
NOME VARCHAR(100)                ID_USUARIO VARCHAR(5) FK                    CATEGORIA VARCHAR(100)
SENHA VARCHAR(8)                 TAREFA VARCHAR(150)
                                 URGENCIA BIGINT FK
                                 COMPLETA BIT(1)


MENU PRINCIPAL

NO MENU PRINCIPAL TEREMOS AS OPÇÕES
ENTRAR
CADASTRAR
SAIR (ENCERRAR APLICAÇÃO)

MENU DE CONTA

NO MENU DE CONTA TEREMOS AS OPÇÕES:
LISTAR TODAS TAREFAS (REALIZADAS OU AINDA NÃO REALIZADAS)
MARCAR TAREFA COMO FEITA (SUB-MENU)
ATUALIZAR TAREFA (SUB-MENU)
REMOVER TAREFA (SUB-MENU)
SAIR (SAIR DA CONTA)

SUB-MENU MARCAR TAREFA:
TEREMOS AS OPÇÕES DE 
MARCAR UMA TAREFA
MARCAR DUAS TAREFAS
MARCAR TRÊS TAREFAS
MARCAR TODAS TAREFAS
VOLTAR AO MENU DE CONTA

SUB-MENU ATUALIZAR TAREFA:
ATUALIZAR NOME DA TAREFA
ATUALIZAR URGÊNCIA DA TAREFA

SUB-MENU REMOVER TAREFA:
REMOVER UMA TAREFA
REMOVER TODAS TAREFAS
